1. NPM Introduction
NPM stands for Node Package Manager . It is package manager for JavaScript , which means it is tool to manage libraries , frameworks and other software packages written in JS .
2. Command Line Interface
NPM is primarily used through command line interface . Developers can use various commands to interact with the package manager such as npm install , npm unistall , npm update etc
3.Dependency management : 
NPM helps to manage dependencies . When you install a package . NPM also installs any other packages that it depends on and so on . This ensures that your project has all the necessary components to function correctly.

API(Application Programming Interface):
API stands for Application Programming Interface
It is set of rules and protocols that allows software applications to communicate and interact with each other.
API's are used to enable integration of different systems or services allowing them to work seamlessly.
